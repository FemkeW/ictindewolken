Blok 1 2019

Voor het blok tussen de herfstvakantie en de kerstvakantie 2019 ga ik een eigen project doen: Gamedevelopment in Python.
De reden dat ik graag een eigen project en niet een van de gegeven projecten wil doen, is omdat ik vorig jaar tijdens het gamedevelopment-0 project erachter ben gekomen dat ik software schrijven heel leuk vind en dat wil ik graag meer doen.
Het enige project dit blok waarbij je dit doet, heb ik dus vorig jaar gedaan en het zou niet nuttig of uitdagend zijn omdat opnieuw te doen.
Toen ik de Python-0 module deed, was het idee om een simpel spel te maken, maar ik kreeg al snel zoveel ideeen dat het te veel was om als module te doen.
Met dit project wil ik dus het stukje code dat ik voor de module had gemaakt uitwerken tot een groter geheel en zoveel mogelijk van mijn ideeen ten uitvoer brengen.

Het concept

Het idee is om een text-based roleplaying game te maken waarbij de speler steeds een keuze krijgt tussen een aantal acties, en zo invloed heeft op de verhaallijn van het spel.
Voor de module heb ik een stuk code geschreven die de speler de mogelijkheid geeft zich tussen de locaties in het spel te bewegen, wat qua beschrijvingen van de locaties ook nog moet worden uitgewerkt.
Verder heb ik het idee van een aantal verhalen die ik wil uitwerken en moet de speler natuurlijk verschillende acties kunnen ondernemen.
Wat ik denk dat daarna ook nog mogelijk moet zijn, is het aantal locaties en de verhaallijnen die daarbij horen uitbreiden en dingen toevoegen als wapens en vijanden om tegen te vechten.
Ik vind het moeilijk om in te schatten hoeveel tijd het bovenstaande kost, maar ik heb anders nog meer ideeen die ik zou kunnen uitwerken.
Ik zal de code hiervan schrijven in Python. Hiervoor heb ik Python gedownload op mijn laptop

Begeleiding

Bij dit project ga ik niet of nauwelijks begeleiding nodig hebben, want het meeste kan ik opzoeken en als ik ergens niet uitkom, kan ik met Zazi Nieuwenhuijs overleggen, want zij gaat ook dit project doen, maar we maken elk onze eigen game.

Voortgang: 4 december 2019

We zijn nu over de helft van het project en de basis van mijn spel staat al een tijdje stevig: je kan niet alleen tussen alle locaties heen en weer bewegen, maar ook dingen vinden en acties uitvoeren.
Ook kun je de meeste muizen al vinden en zijn de kleinste verhaallijnen al uitgewerkt. Dit betekent wel dat de twee grootste verhaallijnen nog niet uit zijn gewerkt, evenals een paar kleine, en dat daar dus mijn belangrijkste focuspunten liggen.
Het programmeren zelf gaat eigenlijk best goed, maar ik vind het af en toe nog lastig de code zo efficient mogelijk op te schrijven. In een aantal gevallen is dat wel al gelukt, maar volgens mijn zijn er een aantal functies eigenlijk nog onnodig.
Als geheugensteuntje en voor overzicht teken ik een stroomdiagram van mijn locaties en maak ik een lijstje van wat ik nog wil doen. Dit doe ik vaak opnieuw, omdat ik naarmate ik het meer gebruik, het steeds rommeliger wordt.
