Blok 1 2019

Voor het blok tussen de herfstvakantie en de kerstvakantie 2019 ga ik een eigen project doen: Gamedevelopment in Python.
Het idee is om een text-based roleplaying game te maken waarbij de speler steeds een keuze krijgt tussen een aantal acties, en zo invloed heeft op de verhaallijn van het spel.
Ik zal de code hiervan schrijven in Python. Hiervoor heb ik Python gedownload op mijn laptop.
Vorig jaar met het project Gamedevelopment-0 begonnen we met een praper prototype.
Bij dit project zal ik iets soortgelijks gebruiken, namelijk stroomdiagrammen om duidelijkheid te krijgen over welke keuzes er zijn en waar die toe leiden.
Ik heb voor de Python-0 module al een begin gemaakt en ik zal het tot een veel groter spel uitwerken in dit project.
Bij dit project ga ik niet of nauwelijks begeleiding nodig hebben, want het meeste kan ik opzoeken en als ik ergens niet uitkom, kan ik met Zazi Nieuwenhuijs overleggen, want zij gaat ook dit project doen, maar we maken elk onze eigen game.
