Big Data: Amsterdam in Kleur

Dit blok heb ik gwerkt op kaggle.com.
Je kan mijn werk daar zien in het notebook BBGA datavisualisatie.
Voor dit project heb ik gewerkt met twee datasets van OIS, waaronder de BBGA, om verschillende verhoudingen van verschillende variabelen tussen de stadsdelen van Amsterdam
te visualiseren en in kaart te brengen. Uiteindelijk ben ik op een stuk code uitgekomen waarmee je met behulp van de datasets met verschillende jaartallen
en variabelen choropleths kan maken.
