Overzicht

Dit project begon ik met het ontcijferen van de code van snaip-a om daarmee het principe van ‘Q-learning’ te achterhalen.
Daarna heb ik met Theyn Kan geprobeerd een programma te schrijven dat het spel minesweeper bovenmenselijk kan spelen.
Daarnaast heb ik zelf de cursus Elements of AI gevolgd, omdat ik kunstmatige intelligentie een erg interessant onderwerp en er meer over wilde weten.

Ik zal hieronder eerst een verslag geven van het project dat ik met Theyn heb gedaan en daarna
een overzicht geven van de dingen die ik heb geleerd over Q-learning en kunstmatige intelligentie van snaip-a en Elements of AI.

Minesweeper AI

Het doel van dit project was om een AI te maken die met behulp van machine learning een spel bovenmenselijk leert spelen.

Theyn kende het spel een stuk beter dan ik en bedacht daarom het principe van het programma.
Ik heb dat vervolgens omgezet in code in Python met onder andere de library pyautogui.
We hebben samen de code getest en overlegden constant. De samenwerking verliep goed.

Met pyautogui kun je het programma plaatjes laten detecteren op het scherm de muis laten bewegen. Er zitten ook functies in waarmee het programma met de muis kan klikken.
Klik hier (link) voor de uiteindelijke code.

Het eindresultaat kan minesweeper spelen, maar niet bovenmenselijk: het is behoorlijk traag.
We hebben dit gedurende het project enigszins kunnen verbeteren door ervoor te zorgen dat het programma alleen naar het gebied kijkt van het spel in plaats van elke keer het hele scherm te scannen,
maar nog steeds is het programma erg langzaam.
Dit komt onder andere doordat het steeds het hele speelveld scant en daarom ook allemaal dingen langsgaat die hij eigenlijk zou kunnen negeren.
Ik heb erover nagedacht hoe ik dit zou kunnen aanpassen, maar ben niet op een goede oplossing gekomen.
We hebben geen machine learning toegepast, omdat we niet wisten hoe en dat ook voor dit spel erg ingewikkeld bleek.
Achteraf gezien hadden we een simpeler spel moeten kiezen. Misschien dat het dan wel was gelukt om het doel te behalen.
Desondanks ben ik wel tevreden met het eindresultaat, dat het spel in de meeste gevallen wint (soms hangt het af van geluk en dan kan het programma nog wel eens verliezen).
